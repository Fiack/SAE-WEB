  • Programmeur

    Le programmeur est chargé de transformer un cahier des charges en lignes de code dans un langage de programmation en fonction des besoins des futurs utilisateurs. Il doit être capable d'utiliser beaucoup de langage différent tel que C++, C#, java, python … Après avoir réalisé le programme, il s'occupe également dès maintenances.

  • Développeur de jeux vidéo

    Le développeur de jeux vidéo est un développeur spécialisé dans les jeux vidéo. Il travaille généralement au sein d'une équipe qui a pour rôle de créer le jeu vidéo de A à Z. Il doit être capable de produire des codes rapides et performants. Une fois le jeu vidéo terminé, il peut s'occuper de régler les éventuels problèmes mais aussi de faire évoluer le jeu.

  • Développeur d'application mobile

    Le développeur d'application mobile est un développeur spécialisé dans la conception des logiciels pour mobiles, smartphones et tablettes. Il doit savoir travailler sur différents supports ( Android, Apple …). Il peut aussi adapter des sites web existants en format mobile.

  • Développeur web

    Le développeur web est un développeur spécialisé dans la réalisation d'un site internet et des fonctionnalités qui vont avec. Il doit analyser les besoins, afin de créer un interface web ergonomique, simple, afin de permettre une navigation aisée aux visiteurs. Il doit avoir les connaissances requises dans les langages du web ( HTML, CSS, JavaScript, PHP …) qui lui permettront de s'adapter aux mieux à la situation.

  • Webmaster

    Le webmaster est chargé d'un site web au quotidien, il se charge d'ajouter les derniers contenues , analyser les statistiques et même résoudre les bugs qu'il pourrait s'y trouver. Un webmaster peut avoir des tâches techniques ( ajouter fonctionnalités, gérer bases de données du site … ) mais aussi des tâches éditoriales ( créer contenue, optimiser contenue pour référencement … ).
